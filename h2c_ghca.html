GitHubClassroomのアサインメントリポジトリのクローン方法

How2clone GHC Assignment

リポジトリの作成とクローン / Creating & Clone new Repo

招待URLのリンクを踏むと以下のページにアクセス可能

  1. 既存の（今まで使っていた）GitHubアカウントでログインする
  2. "Authorize GitHub"（ログインしたアカウントに権限付与）をクリック
  3. "Accept this Assignment"（アサインメントの受け入れ）をクリック
  4. 新規Repositoryの作成完了
  5. 以下のメール（2通）が届く
    • Subject: A first-party GitHub OAuth application has been added to your account
    • Subject: xtd-shira invited you to KUAD-XtechDesign/studyprogress
  6. ちょっと戻って"Your assignment has been created here"以降をクリック
  7. "Set up in Desktop"をクリック
  8. GitHubDesktopが（自動で）開き、以下のウィンドウでCloneをクリック
    • RepositoryURL~が間違っていないか確認( https://github.com/KUAD-XtechDesign/studyprogress-yourname.git )
    • LocalPathはローカルPCに保存（クローン）されるフォルダの場所（要確認）
  9. 指定したフォルダ内にディレクトリができていればクローン完了！

GitHubPagesを利用したリポジトリの公開方法

How2publish new Repo by GHP

前準備

公開設定をする前に、
リポジトリにデータをアップロード（プッシュ）しておく

  1. 以下をダウンロード

    2. Example Repository

  2. zipファイルを解凍して、中の"xtda1_dummy"を先ほどクローンしたディレクトリに移動
  3. GitHubDesktopを使って、Commit(コミット)して、Push（プッシュ）する

GitHubPagesの設定 / Setting4GHP

  1. 自分のリポジトリページにいく
  2. "Setting"をクリック
  3. 下の方の"GitHub Pages"の項目を見つける
  4. "Source"のセレクトボックスをクリックし、"master branch"を選択
  5. これで公開設定は完了
