テーマ / Theme

ステークホルダーを意識したプロトタイピング手法の実践

授業の概要と目的 / Lecture Purpose

デザインフィールドにおいて公益性は大きな比重も持ったキーワードです。
本演習は、デザイン対象を取り巻くステークホルダー（利害関係者）を分析し、その結果を踏まえて具体的な形やサービスに落とし込むケーススタディを行います。
実務的なやり取りや制作プロセスをベースに、様々な尺度を用いた判断を多く積み重ねることを目的とします。

到達目標

  • 調査結果から設計要件の抽出ができるようになる
  • 設計要件の優先順位をつけられるようになる
